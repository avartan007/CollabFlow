## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/avartan007/CollabFlow.git

cd CollabFlow
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:5173
```
